# Portfolio Professionnel - Sidiki Coulibaly

## 🚀 Aperçu

Portfolio professionnel moderne et responsive pour Sidiki Coulibaly, IT Support Analyst et IT Infrastructure Engineer. Ce portfolio présente de manière élégante les compétences, expériences et projets de Sidiki dans le domaine de l'infrastructure IT.

## ✨ Fonctionnalités

### 🎨 Design Professionnel
- Interface moderne et élégante
- Animations fluides et transitions
- Design responsive pour tous les appareils
- Thème sombre/clair avec basculement automatique
- Effets visuels avancés (shimmer, hover effects)

### 📱 Responsive Design
- Optimisé pour mobile, tablette et desktop
- Navigation mobile avec menu hamburger
- Gestes tactiles pour mobile
- Support clavier complet (navigation ESC, Tab)

### 🛠️ Technologies Utilisées
- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Styling**: CSS Grid, Flexbox, CSS Variables
- **Animations**: CSS Animations, Transitions
- **Icons**: Font Awesome 6.4.0
- **Performance**: Optimisations de scroll, requestAnimationFrame

### 📋 Sections du Portfolio

1. **Hero Section** - Présentation avec terminal animé
2. **About** - Description professionnelle et statistiques
3. **Skills** - Compétences techniques avec barres de progression
4. **Experience** - Timeline des expériences professionnelles
5. **Projects** - Projets clés avec modales détaillées
6. **Certifications** - Certifications professionnelles
7. **Contact** - Formulaire de contact et informations

## 🎯 Projets Présentés

### 1. Firewall FortiGate avec Sandboxing et VPN SSL
- **Contexte**: Remplacement d'un firewall obsolète
- **Technologies**: FortiGate 600E, FortiSandbox, NinjaOne, vSphere
- **Résultats**: Blocage 100% des ransomwares, délai de traitement ↓ à 2s

### 2. Active Directory Haute Disponibilité avec vSphere et Veeam
- **Contexte**: Migration AD Windows Server 2012 → 2022
- **Technologies**: vSphere, Veeam, PURE Storage, Windows Server 2022
- **Résultats**: Temps de login ↓ à 5s, RTO AD = 10 min

### 3. Virtualisation Haute Performance avec vSphere/PURE Storage
- **Contexte**: Migration de 12 serveurs physiques vers virtualisé
- **Technologies**: vSphere 8.0, PURE Storage, UCS Cisco, SRM
- **Résultats**: Performance ↑ 150%, RTO = 20 min pour VMs critiques

### 4. Monitoring et Surveillance Infrastructure
- **Contexte**: Système de monitoring complet
- **Technologies**: NinjaOne, vCenter, Syslog, Veeam
- **Résultats**: Visibilité complète, réduction 70% du temps de détection

## 🚀 Installation et Utilisation

### Prérequis
- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Serveur web local (optionnel pour développement)

### Installation
1. Cloner ou télécharger le projet
2. Ouvrir `index.html` dans un navigateur
3. Ou servir via un serveur web local

### Développement
```bash
# Servir avec un serveur local simple
python -m http.server 8000
# ou
npx serve .
```

## 🎨 Personnalisation

### Couleurs
Les couleurs sont définies dans `styles/main.css` via des variables CSS :
```css
:root {
  --primary-color: #2563eb;
  --accent-color: #f59e0b;
  --text-primary: #1f2937;
  --text-secondary: #6b7280;
  --background-color: #ffffff;
  --background-light: #f9fafb;
}
```

### Contenu
- Modifier les informations personnelles dans `index.html`
- Ajouter/modifier des projets dans la section Projects
- Personnaliser les compétences dans la section Skills
